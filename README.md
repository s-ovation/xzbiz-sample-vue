# xzbiz-sample-vue

## はじめに

./src/setting.example.ts をコピーし setting.ts にリネームしてください。

setting.ts を開き "API_KEY", "ENC_EC_USER_ID", "SITE_NAME" を設定してください。

モバイルのみで表示されるため chrome 開発者ツールのモバイルモードなどでご確認ください。
